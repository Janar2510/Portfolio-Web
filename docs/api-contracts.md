# API Contracts

## Overview
This document outlines the contracts between the frontend and backend services (Supabase/API routes).

## CRM API
### Contacts
- `GET /api/crm/contacts`: Fetch all contacts.
- `POST /api/crm/contacts`: Create a new contact.

## Builder API
### Sites
- `GET /api/portfolio/sites`: Fetch user sites.
- `POST /api/portfolio/sites`: Create a new portfolio site.
